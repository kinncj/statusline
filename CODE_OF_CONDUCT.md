# Code of Conduct

This project follows the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, focus on the work. Harassment,
personal attacks, and gatekeeping aren't welcome.

## Reporting

If something needs attention, email **kinncj@protonmail.com**. Reports are
read by the maintainer and not shared outside that context unless the
reporter explicitly asks them to be.

The full text of the Contributor Covenant 2.1 governs this project; the
canonical version is at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
